# Коэффициент корреляции
Коэффициент корреляции (r) измеряет силу и направление линейной связи между двумя переменными
**Свойства:**
- Принимает значения от -1 до +1
- Показывает силу и направление связи
- При r = 0 линейной связи нет
- При |r| = 1 связь функциональная
1. Инвариантность к линейным преобразованиям.
2. Симметричность: r(X, Y) = r(Y, X).
3. Оценка тесноты связи, но не причинности.

# Ранговая корреляция
Измеряет связь между ранжированными данными.
- Используется для порядковых данных
- Не требует нормального распределения
- Устойчива к выбросам
ρ = 1 - (6Σd²) / (n(n² - 1)),
- **Поле корреляции**: графическое изображение зависимости

 Зависимость текущих значений ряда от предыдущих. Рассчитывается с помощью коэффициента автокорреляции: rₖ = Σ((yᵢ - ȳ)(yᵢ₊ₖ - ȳ)) / Σ(yᵢ - ȳ)².

# Выборочная дисперсия и среднеквадратичное отклонение
- **Выборочная дисперсия**: s² = Σ(xi - x̄)²/(n-1), x̄ – выборочное среднее, n – объем выборки.
- **Среднеквадратичное отклонение**: s = √s²
- Характеризуют разброс данных относительно среднего

# Связь t-критерия и частных F-критериев
- F = t²
- t-критерий используется для проверки значимости одного параметра регрессии
- F-критерий – для оценки модели в целом

# Коэффициенты частной корреляции
- Измеряют связь между двумя переменными при фиксированных остальных
- Частный F-критерий проверяет значимость частных коэффициентов

# Индекс детерминации нелинейных моделей
- R² = 1 - (Σ(yi - ŷi)²)/(Σ(yi - ȳ)²)
- Показывает долю объясненной дисперсии
- Принимает значения от 0 до 1

# Доверительные интервалы
- Доверительный интервал (ДИ) – диапазон, в котором с заданной вероятностью находится истинное значение параметра.
- Ширина зависит от уровня значимости
- Формула: оценка ± (критическое значение × стандартная ошибка)
- x̄ ± t * (s / √n), где x̄ – выборочное среднее, t – квантиль t-распределения, s – стандартное отклонение, n – объем выборки.
- Если ДИ не включает значение, предполагаемое нулевой гипотезой, гипотеза отвергается.

# Частная и парная корреляция
- **Парная**: между двумя переменными
- r = Σ(xᵢ - x̄)(yᵢ - ȳ) / √[Σ(xᵢ - x̄)² * Σ(yᵢ - ȳ)²].
- **Частная**: между двумя при фиксированных остальных
- rₓᵧ⋅z = (rₓᵧ - rₓz * rᵧz) / √[(1 - rₓz²)(1 - rᵧz²)].
- Расчет через определители матрицы корреляций
# Рассчитайте линейный коэффициент корреляции.
Формула линейного коэффициента корреляции:
r = Σ(xᵢ - x̄)(yᵢ - ȳ) / √[Σ(xᵢ - x̄)² * Σ(yᵢ - ȳ)²].
Проверка значимости:
t = r√(n - 2) / √(1 - r²),
где n – число наблюдений. Сравнить t с критическим значением из таблицы распределения Стьюдента.

# Метод скользящих средних
**Выравнивание ряда динамики** помогает сгладить краткосрочные колебания и выявить тренд.
**Метод скользящих средних**:
1. Определяется окно (длина интервала, например, 3 или 5 наблюдений).
2. Для каждого интервала вычисляется среднее.
3. Средние значения отображаются на графике.

- Формула: yt = (yt-k + ... + yt + ... + yt+k)/(2k+1)
- k - полуширина окна сглаживания
- Пример для окна 3
- MA₁ = (y₁ + y₂ + y₃) / 3,
- MA₂ = (y₂ + y₃ + y₄) / 3

# Экспоненциальное сглаживание
Метод экспоненциального сглаживания придает разный вес значениям: более новые данные получают больший вес. Формула:
- Sₜ = αyₜ + (1 - α)Sₜ₋₁, где Sₜ – сглаженное значение в момент времени t, yₜ – фактическое значение, α – коэффициент сглаживания (0 < α < 1).

# Нормальное распределение
Нормальное распределение – симметричное распределение, где значения сгруппированы вокруг среднего:
- Симметричное колоколообразное
- Определяется μ и σ
- Правило 3σ охватывает 99.7% наблюдений
При обработке данных:
1. Проверяется нормальность выборки (тест Шапиро-Уилка, критерий Андерсона-Дарлинга).
2. Если данные распределены нормально, применяются параметрические методы анализа (например, t-критерий).

# Критерий Хи-квадрат
- Проверяет согласие эмпирических и теоретических частот
- χ² = Σ((O-E)²/E)
- O - наблюдаемые частоты
- E - ожидаемые частоты
- Если χ² > критического значения, гипотеза отвергается.

# Линейная парная регрессия

- b - коэффициент регрессии
- a - свободный член
- Оценка параметров через МНК
1. Найти коэффициенты регрессии:
2. b = Σ(xᵢ - x̄)(yᵢ - ȳ) / Σ(xᵢ - x̄)²,
3. a = ȳ - b * x̄,
4.  Построить регрессионное уравнение.
5. Проверить значимость коэффициентов (t-критерий) и модели в целом (F-критерий).

# Генеральная совокупность и выборка
- **Генеральная**: все возможные элементы
- **Выборка**: подмножество генеральной совокупности
- **Характеристики**: средние, дисперсии, доли
- Среднее генеральное: μ = ΣX / N.
- Среднее выборочное: x̄ = Σxᵢ / n.
- Генеральная дисперсия: σ² = Σ(Xᵢ - μ)² / N.
- Выборочная дисперсия: s² = Σ(xᵢ - x̄)² / (n - 1).

# Оценки параметров
- **Несмещенная** – ожидаемое значение оценки равно истинному параметру: E(θ̂) = θ
- **Смещенная** – систематическое отклонение от истинного значения: E(θ̂) ≠ θ
- **Состоятельная**: сходится к истинному значению при n→∞
- **Эффективная**: имеет наименьшую дисперсию

2. Рассчитать середины интервалов для каждой группы.
3. Построить точки с координатами (середина интервала, fᵢ).

# Корреляционная связь X и Y
1. Построить диаграмму рассеяния.
2. Рассчитать коэффициент корреляции (r) для определения тесноты связи.
3. Построить регрессионное уравнение (Y = a + bX).
4. Нанести линию регрессии на диаграмму.
# Коэффициенты уравнения регрессии
Уравнение регрессии: Y = a + bX.
1. Найти наклон (b): b = Σ(xᵢ - x̄)(yᵢ - ȳ) / Σ(xᵢ - x̄)².
2. Найти свободный член (a): a = ȳ - b * x̄.

# Числовые характеристики ряда
- Средняя арифметическая - x̄ = Σxᵢ / n.
- Среднее квадратическое отклонение - s² = Σ(xᵢ - x̄)² / (n - 1).
- Дисперсия - s = √s².
- Коэффициент вариации (V = s/x̄ × 100%)
- низкий коэффициент вариации (CV < 10%) указывает на низкую изменчивость данных, высокий – на значительную.

# Множественная регрессия
- **Уравнение**: Y = b₀ + b₁X₁ + b₂X₂ + … + bₖXₖ,
- **Стандартизованное уравнение**:Все переменные переводятся в стандартизованную форму (z-оценки). Уравнение становится:
  Zᵧ = β₁Zₓ₁ + β₂Zₓ₂ + … + βₖZₓₖ.
- **Индекс корреляции**: R = √(1 - ∑(yi - ŷi)²/∑(yi - ȳ)²)
- **Скорректированный R²**: учитывает число параметров
- R²_adj = 1 - [(1 - R²)(n - 1) / (n - k - 1)]
- **Индекс детерминации**: R² показывает долю объясненной вариации
